# Flipkart Customer Satisfaction Prediction

This project analyzes Flipkart customer support data to understand key drivers of customer satisfaction (CSAT) and build a machine learning model that predicts whether a customer will be satisfied or dissatisfied.

## 🔍 Project Overview
- Performed extensive **Exploratory Data Analysis (EDA)**  
- Cleaned and preprocessed customer interaction, product, and agent performance data  
- Engineered meaningful features  
- Built multiple ML models and selected the best-performing one  
- Tuned hyperparameters for improved accuracy  
- Saved the final model for deployment  

## 📊 Tasks Performed
- Handling missing values & outliers  
- Encoding categorical columns  
- Text preprocessing for remarks  
- Balancing dataset using oversampling (SMOTE)  
- Model comparison: Logistic Regression, Random Forest, XGBoost  
- Model explainability using feature importance & SHAP  

## 🧠 Final Model
**Random Forest Classifier (Tuned)**  
- High accuracy, precision & recall  
- Best at identifying dissatisfied customers  
- Deployment-ready with saved `.pkl` model  

## 📁 Files Included
- `Sample_EDA_Submission_Template.ipynb` – EDA notebook  
- `Sample_ML_Submission_Template.ipynb` – ML modeling notebook  

## 🚀 Tech Stack
- Python  
- Pandas, NumPy  
- Matplotlib, Seaborn  
- Scikit-learn  
- SHAP  

## 📝 Conclusion
The project successfully identifies important factors affecting customer satisfaction and provides a robust ML model that Flipkart can use to enhance support efficiency, reduce dissatisfaction, and boost customer retention.
